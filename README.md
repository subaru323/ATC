# ATC エコクイズ

ATCグリーンエコプラザ向けの手認識インタラクティブクイズシステム

## 概要
MediaPipe Handsを使用した手認識により、画面に触れずに環境クイズに答えられる体験型コンテンツ

## ターゲット
- 小学生〜中学生
- 親子連れ（母親世代）

## 技術スタック
- MediaPipe Hands（手認識）
- HTML5 Canvas
- Vanilla JavaScript

## 機能
- カメラによる手認識
- 5問の環境クイズ
- 正解時のパーティクルエフェクト
- スコア表示
- 結果画面

## 使い方
1. ブラウザでindex.htmlを開く
2. カメラアクセスを許可
3. 人差し指で選択肢をタッチ
4. 全問クリアを目指す
